### Gemini project prompt 양식
==================================================================================================
### 1. 구현 목표
게시글 작성 시 '비밀글' 체크박스를 만들고, 비밀글은 권한이 있는 사람만 볼 수 있게 하고 싶어.

### 2. 구체적인 요구사항
1. 글쓰기 화면(write.html)에 '비밀글 여부(secretYn)' 체크박스 추가.
2. DB의 board 테이블에는 이미 secret_yn 컬럼이 있는 것 같아.
3. 리스트 화면(list.html)에서는 비밀글일 경우 제목 옆에 '잠금 아이콘' 표시.
4. 비밀글 클릭 시:
   - 작성자 본인이거나 관리자(ADMIN)면 내용 표시.
   - 권한이 없으면 "작성자와 관리자만 볼 수 있습니다." 알림창 띄우고(UiUtils 사용) 리스트로 이동.

### 3. 관련 파일
- BoardController, write.html, list.html, BoardDTO

### 4. 기타 요청 사항
- BoardController에서 권한 체크하는 로직을 상세하게 짜줘.
